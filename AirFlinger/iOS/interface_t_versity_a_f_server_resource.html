Detailed Description

This object contains the information about a single resouce of a media item returned by a media server browse request.

Generally you will not create such objects yourself, but get them in the block returned by calling browseChildrenWithBlock:ofObject:startIndex:fetchCount:error: (TVersityAFServer)
