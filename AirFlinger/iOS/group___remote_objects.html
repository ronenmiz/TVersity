Detailed Description

Classes representing objects on a remote media server.

The classes in this module represent media items or containers on a remote media server.

Defines the block to be invoked with the response to a request to get children of a container object.

Parameters
containerthe container on which browse was requested, filled with its child objects.
isErrorYES if an error occored. Otherwise NO.
errornil, unless an Error occured.
The list of possible errors that can be returned by this block is found in browseChildrenWithBlock method.
