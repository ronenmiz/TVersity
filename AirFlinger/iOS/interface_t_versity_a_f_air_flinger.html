Detailed Description

This is the main AirFlinger class. With this class you start, initialize and terminate the AirFlinger SDK. Through it you will get access to player and server devices identified on the network.

This class should never be instantiated directly. To get an instance use the getInstance method. You must call initInstance method before using any other method of this class.

- (NSArray *) getCurrentDevices: (TVersityAFDeviceKinds)  deviceKind

Get available devices.

Parameters
[in]deviceKindAn optional parameter for getting devices of a specific kind. Use nil to get devices of any kind.
Returns
An array of detetcted device objects, or nil if no device was found. If deviceKind was passed, only devices of the requested kind will be returned. Use the delegate passed to the initInstance method for monitoring changes to this list. Important: once a device is added to the list, it is never removed. Namely, it can turn offline if disconnected or shutdown and then online again.
