Detailed Description

Provides a uniform interface for accessing any kind of media item supported by the OS. When you request a media of a certain type, the provider will get from the OS all the items of that type, regardless of their source. For example if you request item of type movie, the provider will get all movies from the iPod library and from the ALAessts library. The items can be accessed in a uniform interface using the TVersityAFMediaItem class.
