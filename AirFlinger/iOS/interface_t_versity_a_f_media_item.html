Detailed Description

A uniform representation of an iOS media item (asset).

This object can represents media items from the iPod library or from the PhotoAPP. i.e. AVAsset, MPMediaItem or ALAsset.

Method Documentation

+ (TVersityAFMediaItem *) mediaItemWithAsset: (id)  asset

Initialize a new item object with an OS asset object.

Generally you do not need to create a TVersityAFMediaItem yourself. You'll get it from the TVersityAFMediaProvider.

Notice that in case of ALAsset - the TVersityAFMediaItem is only valid as long as the ALAssetLibrary that produced the asset is still valid. Therefore, for ALAsset objects, it is recommended to use mediaItemWithAsset:andLibrary:

+ (TVersityAFMediaItem *) mediaItemWithAsset: (id)  asset
andLibrary: (ALAssetsLibrary *)  library 

Initialize a new item object with an OS asset object and pass a reference to ALAssetsLibrary to ensure a source ALAsset will remain valid during the lifetime of the object.

Generally you do not need to create a TVersityAFMediaItem yourself. You'll get it from the TVersityAFMediaProvider.

Notice - library is optional, relevant only to ALAsset and used just to retain the library so the ALAsset will not become invalid.
